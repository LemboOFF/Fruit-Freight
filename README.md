# Fruit-Freight

Game made by LemboOff on GitHub

Vibe code is responsible by Claude



View your character stats and abilities in the main menu: hint, they're important on how you play with each character!

Use WASD to move and the Arrow Keys to shoot

Kill the boss of your level and press space to use your special ability
